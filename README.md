# meu-projeto

Site com cena 3D interativa usando Spline, Next.js e shadcn/ui.

🔗 **[Ver site ao vivo](https://eoguilhermeferreira.github.io/meu-projeto)**

## Stack

- [Next.js](https://nextjs.org/) 16
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Spline](https://spline.design/) — cenas 3D interativas
- TypeScript

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
